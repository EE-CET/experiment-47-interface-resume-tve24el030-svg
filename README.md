# Experiment 47: Interface (Resume)

## Problem Statement

Create an interface `Resume` with a method `biodata()`.
Implement this interface in a class `Teacher` to display the resume details.

## Input Format

* Line 1: Name (String).
* Line 2: Qualification (String).
* Line 3: Experience (years, integer).

## Output Format

* Name: [Name]
* Qualification: [Qualification]
* Experience: [Experience] years

### Example 1

**Input:**

```text
Mary
MTech
5
```

**Output:**

```text
Name: Mary
Qualification: MTech
Experience: 5 years
```
